O que é Perícia Digital?

A perícia digital (ou forense computacional) é o processo científico de coleta, preservação, análise e apresentação de evidências digitais. Utilizada em investigações criminais, ela permite que especialistas recuperem dados de dispositivos eletrônicos mantendo a integridade jurídica das provas.

Autopsy

Baseado no framework The Sleuth Kit · Open Source · Multiplataforma

O Autopsy é uma plataforma de análise forense digital de código aberto, amplamente utilizada por peritos e investigadores para examinar dados armazenados em dispositivos eletrônicos. Sua interface gráfica facilita a análise de evidências digitais, tornando-o acessível tanto em ambientes profissionais quanto acadêmicos.

Open Source Gratuito Windows / Linux / macOS The Sleuth Kit

Pro-Tip

O Autopsy é usado por forças policiais como o FBI e a Polícia Federal Brasileira. Sua versão gratuita oferece praticamente as mesmas funcionalidades das soluções comerciais.

É a ferramenta de referência no ensino de forense digital em universidades ao redor do mundo.

O que a ferramenta faz

find_in_page
Recuperação de arquivos

Recupera arquivos apagados de discos rígidos, pen drives e imagens forenses, mesmo após formatação.

history
Histórico de atividades

Identifica atividades do usuário: histórico de navegação, e-mails acessados, downloads e programas executados.

category
Categorização automática

Organiza automaticamente os dados coletados, destacando possíveis evidências como imagens suspeitas e padrões de comportamento.

manage_search
Busca por palavras-chave

Indexa e permite buscas por palavras-chave em grandes volumes de dados, facilitando a localização de conteúdo específico.

timeline
Linha do tempo

Gera uma linha do tempo das atividades no dispositivo, ajudando a reconstruir eventos em ordem cronológica.

extension
Módulos e plugins

Suporta módulos adicionais para análise de geolocalização, comunicadores, redes sociais e muito mais.

FTK Imager

Desenvolvido pela Exterro (ex-AccessData) · Gratuito · Windows

O FTK Imager é um software forense desenvolvido pela Exterro, focado principalmente na criação de imagens forenses — cópias exatas de dispositivos de armazenamento. É uma ferramenta leve, gratuita e bastante acessível, frequentemente usada como o primeiro passo em uma investigação digital antes do uso de ferramentas mais completas.

Gratuito Windows Exterro / AccessData Leve e rápido

Por que preservar antes de analisar?

Em qualquer investigação forense, a regra de ouro é nunca analisar o dispositivo original. O FTK Imager cria uma cópia exata, permitindo que o original seja preservado como evidência jurídica intocada.

verified

Validade jurídica

As imagens geradas pelo FTK Imager são aceitas como evidência em processos judiciais por garantirem a integridade dos dados via hash MD5/SHA.

O que a ferramenta faz

content_copy
Criação de imagens forenses

Captura cópias bit a bit de discos rígidos, pen drives e outros dispositivos, preservando cada byte do original.

fingerprint
Verificação por hash

Garante a autenticidade das evidências usando algoritmos MD5 e SHA-1/SHA-256, comprovando que os dados não foram alterados.

folder_open
Visualização sem alteração

Permite navegar por arquivos e pastas dentro das imagens sem modificá-las, mantendo a validade jurídica das informações.

restore_from_trash
Recuperação de deletados

Recupera arquivos que foram deletados do dispositivo, mas ainda não foram sobrescritos no sistema de arquivos.

data_object
Análise de metadados

Extrai metadados de arquivos como data de criação, modificação, autor e localização GPS em fotos.

output
Exportação para análise

Exporta dados relevantes em formatos compatíveis com ferramentas de análise mais completas, como o FTK completo e o Autopsy.

Como seria aplicado no projeto

1
Ferramenta introdutória ao processo forense

O FTK Imager seria apresentado como a porta de entrada ao processo de investigação forense, mostrando a etapa mais crítica: a coleta segura de evidências antes de qualquer análise.

2
Demonstração da coleta segura

Exemplo prático de como é feita a coleta de evidências digitais de forma segura, sem comprometer a integridade dos dados originais — conceito fundamental em qualquer investigação.

3
Explicações passo a passo

Em ambiente educacional, pode ser explorado por meio de simulações ou explicações passo a passo, mostrando o fluxo completo da criação de uma imagem forense.

4
Conceito de cadeia de custódia

Ajuda os usuários a compreenderem a importância da preservação das evidências e o início do processo pericial, incluindo o conceito de cadeia de custódia em crimes digitais.

Uso em conjunto — o fluxo ideal

Na prática, as duas ferramentas se complementam: o FTK Imager é usado primeiro para criar uma imagem forense segura do dispositivo, e o Autopsy é então aplicado sobre essa imagem para a análise detalhada das evidências — sem nunca tocar no dispositivo original.
